# 🌸 Mother's Day Tribute Website

A visually stunning and interactive Mother's Day tribute website built using pure HTML, CSS, and JavaScript.

This project combines elegant typography, smooth animations, floating petals, cinematic transitions, and interactive UI elements to create an emotional and immersive storytelling experience.

---

## ✨ Features

### 🎨 Elegant UI Design
- Luxury-inspired color palette
- Responsive typography
- Glassmorphism cards
- Smooth gradients and overlays

### 🌸 Advanced Animations
- Animated gradient mesh background
- Floating flower petals using HTML5 Canvas
- Sparkle particle effects
- Scroll reveal animations
- Smooth parallax scrolling
- Interactive hover tilt effects

### 💖 Interactive Sections
- Hero landing section
- Wishes gallery cards
- Emotional love letter section
- Treasured photo gallery
- Interactive blooming flower
- Auto-rotating quote carousel

### 📱 Fully Responsive
- Mobile-friendly layout
- Optimized typography scaling
- Adaptive grids for smaller screens

### ♿ Accessibility Features
- Semantic HTML structure
- ARIA labels
- Keyboard-interactive components
- Smooth scrolling navigation

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- Vanilla JavaScript
- HTML5 Canvas API
- Google Fonts
